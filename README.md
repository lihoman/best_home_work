# best_home_work

This repository contains some small instructional projects:

  1. ATM - a project of ATM, which can change your banknote into smaller denominations, provided that the amount is available at the ATM
  
  2. Wallpapers - a project with program, which can calculate how much wallpaper rolls you need to wallpaper a rectangular room including doors and windows

  3. finding_phrase_in_dir_with_books - a program to search files in a certain directory that have overlapping (the same words or phrases)
